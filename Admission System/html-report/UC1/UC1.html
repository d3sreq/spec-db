UC1: Login to the system

  • User opens main page.
  • System presents main page with a login form.
  • User fills the login form with the authentication data.
  • System verifies the given data.
  • System welcomes Candidate.
Extension : 4a. Not all obligatory data was given.
  • System points which data is missing.
  • Go back to step 3. #goto:3
Extension : 4b. No account with the certain login exists in the system.
  • System informs the User that there is no account with the given user name in the system.
  • System suggests the User to register in the system as a Candidate or contact Administrator to create new account.
  • Go back to step 3. #goto:3
Extension : 4b2a. User decided to register as a Candidate.
