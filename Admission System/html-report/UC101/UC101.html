UC101: Register in the system

  • Candidate opens system main-page.
  • Candidate chooses registration option.
  • System presents a registration data form and asks to enter the registration data.
  • Candidate fills the registration data form and submits the registration data form. #open:registration
  • System verifies if data is correct.
  • System informs that account has been created. #close:registration
Extension : 5a. Some obligatory fields were not filled.
  • Systems highlights the missing fields. #close:registration
  • Back to step 4. #goto:4
Extension : 5b. Account with the given user name already exist.
  • System informs that the user nameUserAccount.userName is in use. #close:registration
  • Back to step 4. #goto:4
Extension : 5c. Given passwords don't match.
  • System informs Candidate that passwords don't match. #close:registration
  • Back to step 4. #goto:4
